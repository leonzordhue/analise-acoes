Buscar processo por número
Informe o número no formato CNJ (ex: 0800770-06.2026.8.19.0046) e o tribunal. · Nota: A API DataJud não retorna nome das partes — disponível via Escavador API.

Busca por filtros combinados
Combine tribunal, período, assunto, grau e comarca. Todos os campos são opcionais — use o que fizer sentido para sua análise.
⚠ Use códigos da família 77xx — os códigos 104xx são inconsistentes entre tribunais
