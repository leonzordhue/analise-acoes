O frontend consulta o CNJ DataJud via Google Apps Script como proxy. O backend já está configurado e pronto para uso. Altere os campos abaixo apenas se precisar trocar o deploy.

  2. Implantar → App da Web → Executar como: Eu → Acesso: Qualquer pessoa
  3. Copie a URL de implantação e cole abaixo
  4. Obtenha sua chave API DataJud em datajud-wiki.cnj.jus.br
